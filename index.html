Datenschutzerklärung

JIRA Queue Highlighter speichert oder verarbeitet keinerlei personenbezogene Daten.

Welche Daten werden verarbeitet?

Alle Daten (z. B. farbliche Einstellungen oder Schalter für die Markierungsoptionen) werden ausschließlich lokal im Browser des Nutzers gespeichert. Es erfolgt keine Übertragung an Server oder Dritte.

Zugriffsrechte

Die Erweiterung benötigt Zugriffsrechte auf Webseiteninhalte (Hostberechtigungen), um bestimmte Bereiche innerhalb von JIRA-Instanzen farblich hervorzuheben. Dabei werden jedoch keine Daten gesammelt, gespeichert oder übertragen.

Kontakt

Bei Fragen zur Erweiterung oder zum Datenschutz können Sie sich gerne an uns wenden.
